### 🏥 MedAgent CRM Pro
> `FastAPI` `LangGraph` `Groq Llama 3` `PostgreSQL` `React` `Redux` `Docker` `Railway`

Dual-mode API: structured form + natural language chat, both writing to one PostgreSQL schema with zero data duplication. **60% reduction** in manual logging. Zero-downtime CI/CD on every push.
